# Patroclosh Bot

Este repositorio contiene un **bot de Discord** desarrollado en **TypeScript** utilizando la librería **discord.js**.

El proyecto fue creado con el objetivo de **aprender en profundidad cómo funciona discord.js**, comprender mejor el desarrollo de bots para Discord y contar con un bot propio, **personalizable y adaptado a mis necesidades**.

---

## Características

-   Bot desarrollado en **TypeScript**
-   Uso de **Discord.js v14**
-   Arquitectura modular (comandos y eventos)
-   Soporte para **Slash Commands**
-   Sistema de carga dinámica de comandos
-   Enfoque en buenas prácticas y tipado estricto

---

## Tecnologías utilizadas

-   Node.js
-   TypeScript
-   Discord.js
-   dotenv
